# ge_money
GE Money - Your Personal Money Journal

GE Money membantu Anda mencatat keuangan pribadi sehari-hari secara
sederhana: input transaksi pemasukan/pengeluaran, kelompokkan dengan
kategori dua tingkat, dan pantau saldo tiap rekening.

Fitur utama:
- Manajemen kategori (induk dan sub-kategori)
- Manajemen rekening dan rekonsiliasi saldo
- Input transaksi dengan opsi perulangan (harian/bulanan/tahunan)
- Laporan: sisa saldo per tanggal, kategori terbesar (pie chart),
  dan arus kas harian

Privasi & data:
Seluruh data disimpan secara lokal dalam file JSON di folder "data"
di sebelah aplikasi. Tidak ada data yang dikirim ke mana pun. Untuk
backup, cukup salin folder "data".
